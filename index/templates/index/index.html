My Projects

Liquid Bug Tracker

A bug tracker web-app that keeps track of reported software bugs or issues in software development projects. Helps teams collaborate and get work done easily and more efficiently. The system has a per-project role-based access control for users. The system allows for 4 types of users; Admin, Project Manager, Developer, and Submitter.

Site hosted on Heroku's free plan. Might take a moment to open on first launch.

  1. Django Bootstrap PostgreSQL AWS Heroku

Mzansi News

A news articles website. Gathers all the headlines from all the major news networks. Users can sign up to get access to the site's search functionality, which allows them to search for news based on location, category, or topic. Made use of NEWS-API to make calls for the data.

Runs on Heroku's free plan. Might take a moment to open on first launch.

  1. RESTful API Django Bootstrap Heroku
