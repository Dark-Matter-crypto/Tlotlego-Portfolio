Mzansi News

A hub for all local news. Users can sign up to get access to search functionality which allows them to search for news based on location, category, or topic. I used the free NEWSAPI to make calls for the data. The challenging part of the project was having to convert each of the dates in the dataset dictionary, provided as a string, from the UTC time zone to SAST. Runs on Heroku's free plan. Might take some time to open on first launch.

MGP Weather

A weather website that allows users to search for weather conditions in all major cities. I used the openweathermap.org free API to make calls for the weather. The documentation helped me to get started. Most of the time was exhausted trying to get the weather card design right. Runs on Heroku's free plan. Might take some time to open on first launch.
