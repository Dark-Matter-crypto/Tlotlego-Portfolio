Liquid Bug Tracker

A bug tracker web-app that keeps track of reported software bugs or issues in software development projects. Helps teams collaborate and get work done easily and more efficiently. The system has a per-project role-based access control for users. The system allows for 4 types of users; Admin, Project Manager, Developer, and Submitter.

Site hosted on Heroku's free plan. Might take a moment to open on first launch.

  1. Django PostgreSQL AWS Heroku

Swift Stay

A property management system that helps students easily find and apply for accommodation, and allows agencies to process student applications and manage lease agreements.

This was a 4 people team project. I was mainly responsible for building the RESTFUL API consumed by both our web app and mobile app, as well as the deployment of the system. Additionally, as the team lead, I was responsible for making sure each of our team members understood their assignment and ensuring that we completed the assigned tickets at the end of each week.

  1. Django Rest Framework AWS Google Maps & Places API PostgreSQL Xamarin Heroku
